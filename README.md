# Tasks

## Description
A simple daily task manager app which allows you to track your daily tasks so you can get more things done!

## Tech Stack 
1. Android Studios - IDE
2. Jetpack compose - UI
3. Android Room - Database
4. Kotlin 

App uses MVVM architechture
